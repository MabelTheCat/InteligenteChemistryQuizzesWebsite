About

The Inteligente Chemistry Quizzes app is an app that is completely free to download, share, modify, and use. No account is needed.

  • It works by generating questions automatically, then you type the answers. If you get the wrong answer, the app will display which answer was the right one.
  • The app is highly customisable, and you can even program your own modified version if you want.
  • The app is licensed under CC-BY-SA-4.0.

Questions & Answers

Question: Can I help contribute ideas or code for the app?

Answer: Yes! Check the contribution section.

Question: Can I help contribute ideas or code for the website?

Answer: Yes! You can open an issue on the GitHub repository.

Question: What do I do if I find a problem in the app?

Answer: Contact us! Check the how to report an error section on what to do and how to do it.

Question: What do I do if I do not have a GitHub account? Do I need to make one?

Answer: No, you do not need to make an account to download and use the app! If you want to contact us, you can use one of the alternatives in Contact Us.

Contribution Rules

Only include one suggestion at a time.

What to include (information with a * before it is required)

  • *Base idea or change
  • *Short description - Try to give us a short description so we can get an idea of what your suggestion is about.

Be kind and professional

How to contribute (click to open)

Contribution Rules

Welcome! Contributing to this open source project is a great way to learn and help others. Here's what you need to know:

Before you contribute

  • Read the rules
  • Understand the project - run the app a few times to get a feel for things and read the relavent code.
  • Pick a simple issue - Look for tags like good first issue or beginner friendly if you are new to programming or the project.
  • Ask before starting (optional) - You can comment on an issue saying, "Can I work on this?" to avoid duplicate additions.

How to make changes

  • Fork the project - This makes your own copy of the project to edit.
  • Clone it to your computer - Use a tool like Git or GitHub Desktop to get the files.
  • Make a new branch - Name it something simple like fix-typo or add-feature.
  • Make small, clear changes - Follow the project's style and only change what's needed.
  • Test it - Try to run it, if possible, to make sure it works.

Submit your changes

  • Commit with a clear message - Short and concise, but gives a general idea. Example: fix typo in README.md
  • Push and create a pull request (PR) - This asks the project to accept your changes.
  • Describe what you did - Write a sentence or two explaining your change.
  • Be open to feedback - Maintainers might suggest changes. That's normal and helpful!
  • Your contribution may not be accepted - Sometimes, a contribution might not be a good fit or may need too much fixing. We may not have the time to guide every case — please don't take it personally!

Be kind and professional

  • Be polite - Respect everyone's time and effort. Be friendly in comments.
  • Don't spam - Only send real, helpful changes.
  • Follow the Code of Conduct

How to report an error

How to report an error on the website (click to open)

To report an error on the website, you can contact us with the following information. Information with a * before it is required. Try to give as much on the list as possible.

Information to include:

  • *Problem type (spelling error, translation error, etc)
  • *Problem, including section or exact line
  • *Time visited
How to report an error in the app (click to open)

To report an error in the app, you can contact us with the following information. Information with a * before it is required. Try to give as much on the list as possible.

Information to include:

  • *Problem type (spelling error, translation error, incorrect answer, incorrect explanation, etc)
  • *Problematic question, answer, explanation, etc. If a question or an answer is the problem, include both the question and the answer.
  • *Version used
  • *Language
  • Full list of settings
  • Quiz seed, if you were doing a quiz
  • Question number, if you were doing a quiz
